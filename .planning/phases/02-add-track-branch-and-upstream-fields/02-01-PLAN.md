<objective>
Migrate RepoConfig from the v1 map-based TOML format (`[repos.<name>]`) to the v2 array-of-tables format (`[[repo]]` with required `name` field). Rename the `URL` field to `CloneURL` (TOML: `clone_url`). Keep the in-memory `cfg.Repos map[string]RepoConfig` intact so no caller files need changes.

Purpose: Establishes the v2 `[[repo]]` TOML structure that Plan 02's `track_branch`/`upstream` fields build on. One cohesive breaking migration (precedent from Phase 1's `[workspace]` → `[metarepo]`).

Output: Updated config structs, load/save pipeline, testutil helpers, and all test fixtures.
</objective>
