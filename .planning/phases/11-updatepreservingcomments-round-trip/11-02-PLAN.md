**Table-driven test `TestRoundTrip_PerBlockType`:**

Each case has:
- `name string`
- `initial string` (TOML content of the initial `.gitw` file, with comment lines)
- `setup func(*config.WorkspaceConfig)` (mutations to apply before Save)
- `comment string` (the comment text to assert is present)
- `anchor string` (the TOML key/header that the comment must precede)

Cases to implement (one per block type from diskConfig):

1. **"metarepo"** — Initial file has `[metarepo]` with `# workspace identity` comment above `name`. Mutate: set `cfg.Metarepo.DefaultBranch = "main"`. Assert `# workspace identity` precedes `name`.

2. **"workspace block"** — Initial file has `[[workspace]]` with `# dev workspace` comment before the block. Mutate: add a second `WorkspaceBlock`. Assert `# dev workspace` precedes `name` of first workspace.

3. **"repo"** — Initial file has `[[repo]]` with `# primary repo` comment. Mutate: add a second repo entry. Assert `# primary repo` precedes `name` of first repo.

4. **"repo with branch_override"** — Initial file has `[[repo]]` with `[[repo.branch_override]]` and `# protected branch` comment before the override. Mutate: change repo's `Path`. Assert comment survives.

5. **"remote"** — Initial file has `[[remote]]` with `# github remote` comment. Mutate: add a second remote. Assert `# github remote` precedes `name`.

6. **"remote with branch_rule"** — Initial file has `[[remote]]` with `[[remote.branch_rule]]` and `# block main` comment before the rule. Mutate: change remote's `URL`. Assert comment survives.

7. **"sync_pair"** — Initial file has `[[sync_pair]]` with `# mirror pair` comment. Mutate: add a second sync_pair. Assert comment survives and precedes `from`.

8. **"workstream"** — Initial file has `[[workstream]]` with `# feature stream` comment. Mutate: add a second workstream. Assert comment survives and precedes `name`.

9. **"groups"** — Initial file has `[groups.web]` with `# frontend group` comment. Mutate: add a new group. Assert comment survives and precedes `repos`.

10. **"worktrees"** — Initial file has `[worktrees.infra]` with `# infra worktree` comment. Mutate: add a second worktree. Assert comment survives and precedes `url` or the worktree key.

**Test body for each case:**
```go
s.Run(tc.name, func() {
    dir := s.T().TempDir()
    cfgPath := filepath.Join(dir, ".gitw")
    s.Require().NoError(os.WriteFile(cfgPath, []byte(tc.initial), 0o644))

    cfg, err := config.Load(cfgPath)
    s.Require().NoError(err)

    tc.setup(cfg)

    s.Require().NoError(config.Save(cfgPath, cfg))

    data, err := os.ReadFile(cfgPath)
    s.Require().NoError(err)
    out := string(data)

    s.Assert().Contains(out, tc.comment)
    commentIdx := strings.Index(out, tc.comment)
    anchorIdx := strings.Index(out, tc.anchor)
    s.Assert().GreaterOrEqual(anchorIdx, 0, "anchor %q not found in output", tc.anchor)
    s.Assert().Less(commentIdx, anchorIdx, "comment %q should precede anchor %q", tc.comment, tc.anchor)
})
```
